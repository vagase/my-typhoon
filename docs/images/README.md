## 公开前检查

- 不显示精确住址、经纬度或可识别个人的位置名称；
- 不包含账号、通知、其他应用窗口或私人菜单栏信息；
- 不遮挡或裁掉 Apple 地图及其数据合作方署名；
- 截图中的数据源名称和发布时间保持可见；
- 不把测试数据描述成真实预报；
- 图片使用 PNG，并尽量控制单张文件大小在 1 MB 以内；
- Light/Dark Mode 二选一保持统一，避免同组截图视觉跳变。
