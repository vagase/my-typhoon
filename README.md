# TyphoonBar

一个只驻留在 macOS 菜单栏的台风监测小应用。点击菜单栏的台风图标即可查看：

- 中央气象台最新位置、强度、气压和移动信息
- 浏览近三个月台风，并在菜单中手动切换
- 默认展示当前活动台风；没有活动台风时显示“一切安好”
- 列表优先展示活动台风，同一状态下按最后路径更新时间从新到旧排列
- 根据完整路径计算台风最接近当前位置的日期和距离，辅助判断当地主要影响时间
- 以台风中心向外扩散的百公里风力圈层、典型风级和安全外围
- 按四象限半径绘制的 7 / 10 / 12 级风圈
- 最近实况轨迹与 BABJ 官方未来路径
- 根据 Mac 当前定位计算与台风的最近距离和时间
- 当地未来 24 小时累计雨量、最大阵风和逐小时影响趋势
- 内涝、山洪、强风和户外活动等针对性安全提示
- 每 15 分钟自动刷新，也可手动刷新

## 构建与运行

需要 macOS 14 或更高版本，以及 Xcode Command Line Tools。

```bash
chmod +x build_app.sh
./build_app.sh
open dist/TyphoonBar.app
```

首次启动如果被系统拦截，可在 Finder 中右键应用并选择“打开”。应用采用本机临时签名，不会显示 Dock 图标。

## 数据与限制

实况、路径和分级风圈来自中国气象局中央气象台台风网的公开数据。百公里圈层的典型风级取 Open-Meteo 模式网格中位数，并使用中央气象台 7 / 10 / 12 级风圈校准内圈强度；它不等同于地面测站实况。当地逐时风雨同样使用模式预报并结合台风路径计算影响时段。模式预报不能替代官方预警，实际防灾行动请以当地气象台和应急管理部门发布的信息为准。

应用只把定位坐标用于请求当地天气和在内存中计算影响，不存储位置。首次打开当地影响区域时，macOS 会询问定位权限。
